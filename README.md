# RENZER: Area Codes 🚗

משחק מתנה מצחיק לחבר בשם **רנצר** (שם אמיתי: נווה). הוא נוסע באלפא רומיאו **לבנה** בין בחורות בכל אזורי החיוג של ישראל, בהשראת "Area Codes" של Ludacris. גימיק מרכזי: בארנק שלו יש **בדיוק $5** שלעולם לא זזים — כל העולם משלם בשבילו.

- **אתר ציבורי (לשיתוף):** https://mezereedo.github.io/renzer-area-codes/
- **ארטיפקט claude.ai (אותו קוד):** https://claude.ai/code/artifact/d0e9bb94-5a01-498f-8528-a6af577d4451
- שפה: עברית + אנגלית מעורבב. נוצר ביולי 2026 ע"י Claude Code עבור Oz.

## טכנולוגיה — קובץ אחד בלבד

**כל המשחק הוא `index.html` אחד**, ללא תלויות, ללא build, ללא שרת:

- Canvas 2D בגודל לוגי **520×840** (פורטרייט), מוצג בהתאמה לחלון עם letterbox ו-devicePixelRatio.
- לולאת משחק: `requestAnimationFrame` → `scene.update(dt)` → `scene.draw()`.
- **מערכת סצנות**: משתנה גלובלי `scene` + `setScene(s)` שקורא ל-`s.init()`. כל סצנה היא אובייקט עם `init/update/draw` ואופציונלית `onTap(x,y)` ו-`cheat()`.
- קלט: מקלדת (חצים/WASD/SPACE) + מגע (גרירה = ג'ויסטיק וירטואלי יחסי לנקודת המגע, טאפ קצר = פעולה). `tapped` הוא דגל של פריים אחד; `axisX()/axisY()` מאחדים מקלדת ומגע.
- מוזיקה: **WebAudio מסונתז בקוד** (ראו "מוזיקה" למטה) — אין קבצי אודיו.
- תמונות אמיתיות: מוטמעות כ-**data URI** בקבועים בראש הסקריפט (ראו "תמונות").
- טקסט עברי על קנבס: `ctx.direction='rtl'` לפני ציור, `'ltr'` אחרי. הפונקציות `txt()`, `display()` (כותרות עבות), `bubble()` (בועות דיבור RTL), `rr()` (מלבן מעוגל).

## מבנה הזרימה

```
title → intro (סיפור פתיחה) → INFO(04) → prologue (שדות ים: להגן על הסטאש משירה)
      → winScene(04) → momScene (אמא/יצפן בדלת — קטע קבוע!)
      → map ⟲ עבור כל עיר: drive → [side event?] → INFO(קוד) → minigame → winScene
      → כשכל 5 הערים הושלמו: nofarIntro (שיחה מנופר) → drive לאילת → INFO(077)
      → nofarScene (קריוקי) → nofarWinScene (צילום הוואטסאפ!) → map → ending
```

### הערים והמיני-משחקים (`CITIES` + `MINIGAMES` + `INFO`)

| קוד | עיר | בחורה | מיני-משחק | ניצחון |
|-----|-----|--------|------------|--------|
| 04 | שדות ים (פרולוג) | שירה מהקיבוץ | לחסום אותה מלגנוב אוכל/מים/וויד (top-down) | 6 חסימות לפני 3 גניבות |
| 03 | תל אביב | מאיה, תואר שני במדמ"ח | טיימינג-בר: לדחוף את החשבון אליה | `hits>=need(5)` או `b>=1` |
| 09 | נתניה | אוראל הממנגלת | לתפוס אוכל בצלחת, להתחמק מ-💸/🥗 | 10 תפיסות לפני 3 פשלות |
| 02 | ירושלים | פנינה 💜 (יורדים בגדול) | מבוך דירה חשוכה עם פנס, חתולים, מד חרטה | להגיע למיטה לפני שהמד מתמלא (~42ש') |
| 08 | באר שבע | סטלה המגדלת | אור אדום/ירוק: לגנוב את הצנצנת ולחזור לדלת | בלי להיתפס 3 פעמים |
| 077 | אילת (בונוס) | נופר 🎤 | דואט קריוקי: להחזיק SPACE/אצבע בתוך פס נע | מד קהל > 0.55 אחרי 3 סיבובים |

- כל מיני-משחק מקבל **מסך הוראות** לפניו: `makeInfoScene(city, factory)` + מילון `INFO` לפי קוד. מוסיפים עיר? חובה להוסיף רשומת INFO.
- מסך ניצחון גנרי: `makeWinScene(city, punchline, badgeLine, next?)` — `next` אופציונלי (הפרולוג מעביר ל-`momScene`).
- צ'יט לבדיקות: מקש **`0`** מדלג על המיני-משחק הנוכחי.

### נסיעות (`makeDriveScene(city, startDist?)`)

סקרולר אנכי: 4 נתיבים, מכוניות/משאיות/🐪, פס התקדמות. התנגשות = האטה + שורת בדיחה אקראית על אמינות האלפא (`crashLine`). `startDist` משמש לחידוש נסיעה אחרי קטע צדדי.

### קטעים צדדיים — כלל: **מקסימום אחד בין מיקוד למיקוד**

נבחרים ב-init של הנסיעה (`this.sideEvent`), מדלגים על נסיעות-המשך (`startDist`):

1. **סופר-פארם** (נסיעה 1, חובה, פעם אחת — `S.condoms`): האוטו נעצר בכוח → `pharmScene` — קופה, קונדומים ₪29.90, לרנצר יש $5, הבחורה שהוא בדרך אליה משלמת בביט.
2. **לינה אצל בן** (הנסיעה הראשונה אחרי שת"א הושלמה — `S.bonDone`): `bonScene` — קציצות טונה, פלייסטיישן, "תודה על קציצות הטונה, מלך". התמונה האמיתית של בן מוצגת בעיגול.
3. **אמא (יצפן)** — *לא* קטע נסיעה! סצנה קבועה בשדות ים אחרי ניצחון הפרולוג (`momScene`): "נווה חמוד!", "תעבור בטלסו! שמעתי שיש שם בנות!", "רוצה אבטיח לדרך? 🍉" — והוא מנפנף אותה.

## תמונות (חשוב לשחזור!)

מקורות ב-`photos/` (**לא עולים לגיט** — gitignored לפרטיות). מוטמעים כ-data URI בקבועים בראש `<script>`:

| קבוע | תוכן | שימוש |
|------|------|-------|
| `PHOTO_FACE` | פנים של רנצר (כובע באגס באני + משקפיים), ריבוע 220px | הראש שלו בכל המשחק (`drawRenzer` מצייר קלסתרון מצויר כשהקבוע null) |
| `PHOTO_SIDE` | פרופיל צד | סצנת האינטרו |
| `PHOTO_ENDING` | תמונת הנחל | מסך הסיום |
| `PHOTO_CHAT` | צילום וואטסאפ "יש לנופר גרון מטורף" + חחחח | הפאנץ' אחרי הקריוקי |
| `PHOTO_BEN` | הפנים של בן (מהסירה) | סצנת קציצות הטונה |

תהליך ההטמעה: חיתוך עם `sips` (macOS), למשל הפנים של רנצר: `sips -c 720 720 --cropOffset 560 50 renzer_and_dog.jpeg` ואז הקטנה ל-220px jpeg; ואז סקריפט python שמחליף `const PHOTO_X = null;` ב-data URI. **שימו לב:** אחרי ההטמעה השורות מכילות base64 ענק — עריכות עתידיות צריכות לעגון על טקסט אחר (למשל `const W=520`).

## מוזיקה וסאונד

- ביט מקורי בסגנון dirty-south (מחווה חוקית ל-Area Codes): קיק/סנר/האטים/בס/שריקה מסונתזים ב-WebAudio, scheduler עם lookahead ב-`music.scheduleBar()`. שני מצבים: `menu` (82bpm) ו-`drive` (96bpm + הוק שריקה).
- **אסור להטמיע את ההקלטה המקורית של Ludacris** (זכויות יוצרים). לכן:
  - כפתור 🎧 — המשתמש טוען MP3 מקומי משלו → מפוענח ב-WebAudio ומתנגן בלופ (`music.customBuf`).
  - כפתור ▶️ — נגן יוטיוב רשמי מוטמע בפינה (embed חוקי). **מופיע רק ב-GitHub Pages** — בארטיפקט claude.ai ה-CSP חוסם iframe חיצוני (מוסתר לפי `location.hostname`).
  - כששיר חיצוני מתנגן, `music.update()` משתיק את הביט המובנה.
- אפקטים: `music.sfx('blip'|'good'|'bad'|'win'|'crash'|'meow'|'cash')`.

## מילון בדיחות פנימיות (אל תשנו בלי אישור מ-Oz)

- **רנצר / נווה** — החבר; אמא שלו ("יצפן") קוראת לו "נווה חמוד".
- **טלסו** — המקום שאמא תמיד שולחת אליו ("שמעתי שיש שם בנות").
- **איציק סאסא** — הבעלים האמיתי של הוויד ("אל תיגעי בוויד!! זה של איציק סאסא!!" — בפרולוג).
- **בן** — מארח קציצות הטונה.
- **נופר** — "יש לנופר גרון מטורף" (צילום וואטסאפ אמיתי, מוצג במשחק באישור Oz).
- **איימי** — הכלבה של רנצר (נקבה! לא "הכלב"), יושבת לידו באוטו.
- **יורדים בגדול** — הבדיחה של פנינה מירושלים.
- **האלפא** — לבנה, איטלקית, "זו לא מכה, זה קרקטר."

## כפתורי כוונון מהירים (חיפוש בקוד)

- פרולוג: `blocks>=6`, `steals>=3` | ת"א: `need:5`, זום `zone:0.34`, דריפט `0.03` | נתניה: `score>=10`, `bad<0.24` | ב"ש: `caught>=3` | ירושלים: קצב חרטה `100/42`, מבוך = מערך `WALLS` | נופר: `crowd>0.55`, 3 סיבובים | נסיעה: `dur=20±4s`, קטע צדדי ב-`eventAt=0.42+rnd*0.18`.
